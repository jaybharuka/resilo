## 🏗️ Technical Architecture

### Backend Components
- **`realtime_api_server.py`**: Enhanced FastAPI server with autonomous operations
- **`autonomous_operations_system.py`**: Core autonomous bot with permission management
- **`auth_system.py`**: JWT-based multi-tenant authentication
- **`crisp_chatbot.py`**: Optimized AI chatbot with actionable responses
- **`daily_health_reporter.py`**: Comprehensive health analytics engine
- **`community_help_center.py`**: Full community support system
- **`enhanced_social_bots.py`**: Slack/Discord integration platform
- **`free_apis_integration.py`**: Multi-API aggregation system

### Frontend Components
- **`App.js`**: Enhanced main dashboard with multi-role support
- **`MultiRoleDashboard.js`**: Role-based portal system
- **`DeviceManagementPortal.js`**: Complete device management interface
- **`SimpleSystemMetrics.js`**: Real-time system monitoring
- **`RealtimeChat.js`**: Integrated chat system

### Database Schema
- **Authentication**: Users, companies, sessions, permissions
- **Community**: Tickets, chat messages, knowledge base, forum posts
- **Health**: System metrics, alerts, performance trends
- **Devices**: Inventory, assignments, security status
